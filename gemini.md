# 프로젝트: 운동관리 앱

## 목적
Weight 운동을 진행하며 세트별 무게, 횟수 기록.
운동기록을 받기 전 분할 및 세트 구성 화면 먼저 제공.
현존하는 모든 Weight 운동 종목 및 간략한 그림으로 해당 운동이 무엇인지 볼 수 있어야 함.
분할 방식을 선택하면 기본 운동 루틴은 제공해야함.(인터넷, 논문 검색 필요)
분할 방식 선택 후 사용자가 수정 할 수 있는 창 제공.
운동 수행 능력 증가 현황 제공
다음 수행시 무게, 세트 권장

## 기술 스택
- Backend: FastAPI(Python)
- Frontend: HTML, CSS, JavaScript
- DB : sqlite3 기반의 .db 파일

## 핵심 규칙
모든 루틴은 1주 단위로 구성
각 부위별 스트렝스 1종목 근비대 1종목으로 구성
스트렝스 종목은 총 4세트(1세트 웜업, 2,3세트 탑세트, 4세트 back-off세트 로 구성)
근비대 종목은 총 3세트 모두 같은 무게 횟수를 기준으로 구성
3분할 운동의 경우 가슴 / 등 / 하체어깨 로 구성
2분할 운동의 경우 등가슴 / 하체어깨 로 구성
모든 운동 Day에 이두, 삼두, 어깨 측후면 운동 추가

## 프로젝트 구조
cursor/
-> backend/     # FastAPI 기반 백엔드
-> frontend/    # HTML, CSS, JS 기반 프론트엔드
-> desings/     # 참고 디자인 파일
-> data/        # 데이터베이스 저장파일 (sqlite3 기반의 .db 파일)
